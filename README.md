# 3D Product Viewer

A full-stack MERN application that allows users to upload and interact with 3D models.
Users can rotate, zoom, change materials, apply textures, and customize the viewing environment.

---

## Live Demo

- Frontend: https://3-d-product-viewer-alpha.vercel.app/
- Backend: https://threed-product-viewer-1.onrender.com/
- GitHub: https://github.com/akriti2424/3D-product-viewer

---

## Tech Stack

**Frontend**
- React
- Three.js
- @react-three/fiber
- @react-three/drei
- Tailwind CSS
- Vite

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer
- Cloudinary (for media storage)

**Deployment**
- Vercel (Frontend)
- Render (Backend)

---

## Features

- Upload GLB / GLTF 3D models
- Rotate, zoom, and inspect models
- Change background color
- Toggle wireframe view
- Upload and apply textures
- Switch materials
- HDRI environment lighting
- Store 3D models and media using Cloudinary

---

## API Endpoints

- POST `/api/upload` – Upload 3D model
- POST `/api/settings` – Save viewer settings
- GET `/api/settings` – Fetch viewer settings
